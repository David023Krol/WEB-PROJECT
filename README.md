## **Název projektu**

Auto Bazar luxusních automobilů

## **Žák**

David Krol

## **Stručný popis tématu (3–5 vět)**

AutoBazar+ je moderní jednostránková webová aplikace, která umožňuje uživatelům procházet nabídku ojetých vozů a interaktivně je filtrovat podle různých parametrů. Stránka pracuje s daty uloženými v JSON souboru nebo databázi a pomocí AJAXu načítá výsledky bez nutnosti reloadu stránky. Uživatel si může zobrazit detailní kartu každého automobilu, včetně fotek, technických údajů a popisu. Projekt kombinuje moderní frontend, dynamické filtrování a jednoduchou backendovou komunikaci.

## **Cílová skupina**
